# Rust Utilities

Small utility applications I've developed while learning rust

## Utilities
- `affine-cli`: CLI for the AFFiNE GraphQL API and workspace blob endpoints
- `api-reader`: Read JSON data from an API and write CSV output
- `avahi-scan`: Scan for Avahi services on the local network
- `colcat`: List and print values from collectd sockets

## Running
```bash
  cargo run -p <package_name> -q
```

## Building
```bash
  cargo build --release
```
